# Warr Battle - Premium Mobile eSports Platform

A premium, modern eSports website for the "Warr Battle" mobile gaming app, featuring dark futuristic gaming UI, neon gradients, and glassmorphism design.

## 🎮 Features

### Visual Design
- **Dark Futuristic Gaming UI** - Modern, immersive aesthetic inspired by top esports platforms
- **Neon Gradients** - Purple, blue, and red color scheme with glowing effects
- **Animated Grid Background** - Dynamic, moving grid pattern
- **Glassmorphism Design** - Frosted glass effect with blur and transparency
- **Mobile-First Responsive** - Optimized for all screen sizes

## 📱 Sections

1. **Navigation Bar** - Fixed sticky nav with gradient logo
2. **Hero Section** - Live badge, compelling CTA buttons, animated elements
3. **Features Section** - Daily Tournaments, Instant Withdrawals, Secure Matches, Real Cash Rewards
4. **Games Section** - Free Fire, BGMI, Call of Duty Mobile
5. **Tournaments Section** - Active tournaments with entry fees and prize pools
6. **How It Works** - 4-step process visualization
7. **Leaderboard** - Top 10 players ranking
8. **CTA Section** - Conversion-focused download button
9. **Footer** - Social links, quick links, support info
